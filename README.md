 Introducción 
El Ministerio del Ambiente llevó acabo la evaluación Nacional Forestal cuya implementación ha sido realizada con el apoyo de asistencia técnica y financiera del Programa de Conservación de Bosques y REDD+ (PCB REDD+), Programa Integral Amazónico de Conservación de Bosques y Producción Sostenible (PRO Amazonía) y el Programa REDD Early Movers (REM) en el periodo 2009 al 2013. 
Previo al levantamiento de información las brigadas de campo recibieron capacitación sobre procesos metodológicos, capacidades técnicas para el levantamiento de la información, manejo de GPS, utilización de guías dendrológicas, manejo de herramientas y equipos para medición de árboles para un óptimo proceso de levantamiento de información.  
Una vez ejecutada las actividades antes mencionadas, se ha obtenido un set de datos para caracterizar, cuantificar la existencia de los recursos forestales por tipos de bosques, es decir, el objetivo del plan fue la generación de indicadores, métricas del estado productivo, diversidad y uso potencial de los bosques en el Ecuador. 
Para el presente trabajo, se tomará como variable dependiente o variable objetivo a la “biomasa” de los individuos arbóreos, y así explorar que variables de todo el set de datos tienen una mejor relación con la variable, así como también comprobar si existe evidencia estadística o diferencias estadísticas en los contenidos de Biomasa promedio a nivel de individuos en cada uno de los Estratos de bosque considerados a través del muestreo por el Inventario Nacional Forestal
